Для написания программы использовался python3.4, библиотека PyGame.

Следовательно, чтобы запустить игру, нужно установить python3.4 и PyGame. Игра запускается из консоли командой python3 main.py.

Управление осуществляется мышкой. Игровой процесс начинается по нажатию кнопки "Start" в меню.

Игра представляет собой классический арканоид. Игра начинается с 3 попытками. Если шар улетает под экран, то у игрока становится на одну попытку меньше.

В игре есть несколько видов блоков: обычные, разрущающиеся после одного попадания; блоки с прочностью n, превыщающей единицу, то есть необходимо попасть по ним n раз; блоки с неограниченной прочностью, которые служат препятствием.
Обычные блоки -- серые изображения, а непробиваемые -- черно-желтые.

В игре 5 видов призов:
1.Лишиться 1 попытки (убить палку)
2.Увеличить скорость шара
3.Замедлить скорость шара
4.Получить 1 дополнительную попытку
5.Увеличить размер палки

Между уровнями есть небольшая пауза, чтобы игрок мог осознать происходящее.
